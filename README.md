## Installation

Clone the repository:
```bash
https://github.com/gauravraj347/Simple_Portfolio_Tracker.git
```

Navigate to the frontend directory and install dependencies:

```bash
cd frontend
npm install
npm run dev
```

Navigate to the Backend directory

```bash
cd backend
mvn spring-boot:run

```
